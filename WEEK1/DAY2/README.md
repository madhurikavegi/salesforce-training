# Salesforce Basics & System Design

## 1. What is Salesforce Platform?

Salesforce is a cloud platform used to create and manage business applications online.

### It helps companies:
- Store data
- Manage customers
- Automate work
- Create apps easily

---

# 2. Explain

## App
An **App** is a collection of related features and tabs.

### Examples
- Sales App
- Student Management App

---

## Object
An **Object** stores data like a table.

### Examples
- Student Object → stores student details
- Employee Object → stores employee details

---

## Tab
A **Tab** is used to open and access objects or pages.

### Examples
- Student Tab
- Reports Tab

---

# 3. Difference: Configuration vs Coding

| Configuration | Coding |
|---|---|
| Done using clicks | Done using programming |
| Easy | More complex |
| No coding needed | Coding knowledge needed |
| Faster | Takes more time |

### Examples
- **Configuration** → Creating fields and forms
- **Coding** → Writing Apex programs
